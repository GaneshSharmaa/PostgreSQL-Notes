# PostgreSQL Notes

A structured collection of my PostgreSQL notes, SQL queries, examples, and practice exercises created while learning and revising PostgreSQL from scratch.

I made this in notes format as I keep forgetting the most important stuff. This feeling of "I'm forgetting the core concepts" kicked in when I was doing Database ORM using SQLAlchemy

The goal of this repository is to serve as a quick reference for SQL fundamentals, relational database concepts, and PostgreSQL-specific features.
